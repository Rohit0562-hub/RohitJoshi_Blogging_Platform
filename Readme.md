A simple yet functional blogging platform built using **PHP and MySQL**, featuring user authentication, role-based access, post management, comments, search, and AJAX-powered post loading.

## Features

### User System
- User registration & login
- Secure password hashing
- Session-based authentication
- CSRF protection

### Roles & Permissions
- Normal User
- Post Owner
- Role-based access control

### Blog Posts
- Create, edit, and delete posts
- Assign categories and tags
- Display author name and post date
- Only post owners or admins can edit/delete posts

### Comments
- Logged-in users can comment
- Comments displayed under each post

### Search & UI
- Keyword-based search
- AJAX “Load More” posts
- Responsive and clean layout
